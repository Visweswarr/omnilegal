# OmniLegal v3 — Test Credentials

This project does **not** require user authentication.

All API access is keyed via server-side environment variables stored in
`/app/.env` (Emergent LLM key, Groq, Gemini, Indian Kanoon, CourtListener,
GovInfo, Data.gov, HuggingFace).  The frontend has no login.

For testing, simply navigate to:

  https://omnilegal-precedent.preview.emergentagent.com/

and use any of the 6 pillars (Atlas, Forensics, Advocacy, Live, Council,
Research). No credentials needed.
